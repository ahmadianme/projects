# PCA and K-NN

Dimension reduction using Principle Component Analysis (PCA) for classification using K-Nearest Neighbor (K-NN) on FER2013 dataset.

### Dataset:
FER2013 (Facial Expression Recognition 2013 Dataset) [1].
